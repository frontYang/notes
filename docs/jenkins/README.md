# Jenkins 实现前端自动化工作流

## 安装

在官网[下载地址](https://www.jenkins.io/zh/download/)选择对应的版本及环境下载，我这里选择了windows稳定版安装程序

jenkins-2.222.4

下载完成解压后，有一个`jenkins.msi`文件，直接双击安装即可
